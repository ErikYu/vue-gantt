# vue-gantt

> dhtmlxGantt with vue.js, A gantt component for Vue.js

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```

## gantt功能总结

1. 按钮切换
    - 今天
    - 上一屏
    - 下一屏
    
2. 时间 marker
    - TodayMarker
    - 项目Marker
    
3. 拖拽
    - moveTask
    - 左右拖拽task
    
4. 任务样式
    - 没有开始的时间
    - 没有结束的时间
    - 两者都没有

5. 状态
    - 未开始
    - 进行中
    - 已完成
    - 逾期

6. 点击显示任务弹窗

7. tooltip提示信息

8. 数据
    - 过滤
    - 筛选
    - 格式化
